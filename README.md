# 杜邦座器件与 PCB 封装生成器

这是一个本地静态网页应用，用网格方式生成杜邦母座焊孔阵列，并导出：

- 立创 EDA 标准版封装 JSON
- 立创 EDA 标准版器件 JSON
- KiCad 封装 `.kicad_mod`
- KiCad 器件 `.kicad_sym`

## 使用

直接用浏览器打开 `index.html`。

1. 输入器件名称、高度、宽度、孔距、焊盘和钻孔尺寸。
2. 点击“生成网格”。
3. 点击格子启用或禁用对应杜邦母座焊孔。
4. 选中格子后填写引脚编号和名称。
5. 使用“线 / 矩形 / 文本”模式绘制顶层丝印。
6. 点击导出按钮下载立创 EDA 或 KiCad 文件。

## 立创 EDA 兼容策略

主导出使用立创 EDA 标准版 JSON。封装文件使用标准版 PCB Footprint 文档类型，焊孔使用通孔 `PAD`，丝印使用 `TRACK` 和 `TEXT`。器件文件使用标准版 Schematic Symbol 文档类型。

专业版可优先尝试导入标准版 JSON；如果遇到专业版导入差异，可以使用 KiCad 文件作为备用导入路径。
