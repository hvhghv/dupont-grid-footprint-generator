# 杜邦座器件与 PCB 封装生成器

这是一个本地静态网页应用，用网格方式生成杜邦母座焊孔阵列，并导出：

- KiCad 源文件 ZIP，包含器件库和封装库
- KiCad 封装 `.kicad_mod`
- KiCad 器件 `.kicad_sym`

## 使用

直接用浏览器打开 `index.html`。

1. 输入器件名称、名称丝印字号、高度、宽度、孔距、焊盘和钻孔尺寸。
2. 修改高度或宽度后，中间网格会立即更新。
3. 点击格子启用或禁用对应杜邦母座焊孔。
4. 选中格子后填写引脚编号和名称。
5. 使用“线 / 矩形 / 文本”模式绘制顶层丝印。
6. 选择器件行数、每行引脚数、镜像或翻转，右侧会实时预览 KiCad 符号。
7. 右侧 PCB 预览会同步显示焊盘、钻孔、编号、外框和丝印。
8. 拖动左栏/中间栏、中间栏/右栏之间的分隔条，可以调整三栏宽度。
9. 优先点击“KiCad 源文件 ZIP”，下载同时包含器件和封装的压缩包。

缩小高度或宽度时，如果已有启用格子位于更靠下或更靠右的位置，应用会自动保留能显示这些启用格子的最小行列数。

## 器件样式

当前支持按行组织 KiCad 符号引脚：

- `单行`：所有引脚作为一组从左侧引出
- `双行`：第 1 行在左侧，第 2 行在右侧
- `四行`：按上、下、左、右四侧分组
- `每行引脚数`：留空时自动平均分配，填写数字时按该数量分组，超出的引脚会继续追加行组
- `四行引脚数`：四行模式下可以分别设置上、下、左、右的引脚数量，留空的侧边自动分配剩余引脚
- `器件左右镜像 / 上下翻转`：只改变 KiCad 符号的引脚分布方向，不改变 PCB 焊孔阵列

## PCB 预览

右侧 PCB 预览会显示焊盘、钻孔、引脚编号、自动外框丝印和手绘丝印。

`外框丝印距离 mm` 控制自动外框到最外侧焊盘中心的距离，并同步影响 PCB 预览和导出的 KiCad 封装。

如果勾选 `分别设置外框四边距离`，可以分别填写上、下、左、右四个方向的距离；未勾选时四边使用统一距离。

`名称丝印字号 mm` 控制封装中器件名称文字的丝印字体大小，并同步影响 PCB 预览和导出的 KiCad 封装。

## 自动缓存

应用会在浏览器中自动缓存当前项目数据：

- 每次编辑后会延迟写入 cookie 缓存，下次打开页面会自动恢复上次数据
- cookie 使用分片保存，支持比单个 cookie 更大的项目数据
- 如果浏览器在本地 `file://` 页面下限制 cookie，会自动使用 `localStorage` 作为兜底缓存
- “保存项目 JSON / 导入项目 JSON”仍然可用于手动备份和迁移

缓存内容包含网格尺寸、启用格子、引脚编号和名称、手绘丝印、符号样式、外框距离、镜像和翻转设置。

## KiCad 文件结构

KiCad 源文件 ZIP 内包含：

- `*.kicad_sym`：KiCad 符号库，包含器件引脚
- `*.pretty/*.kicad_mod`：KiCad 封装库目录，包含通孔焊盘和丝印

解压后可以把 `.kicad_sym` 添加为符号库，把 `.pretty` 目录添加为封装库。
