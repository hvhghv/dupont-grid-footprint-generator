# 杜邦座器件与 PCB 封装生成器

这是一个本地静态网页应用，用网格方式生成杜邦母座焊孔阵列，并导出嘉立创EDA专业版库：

- 完整器件库 `.elibz2`，包含器件、原理图符号和 PCB 封装
- 原理图符号库 `.elibz2`
- PCB 封装库 `.elibz2`

## 使用

直接用浏览器打开 `index.html`。

1. 输入器件名称、名称丝印字号、高度、宽度、孔距、焊盘和钻孔尺寸。
2. 修改高度或宽度后，中间网格会立即更新，也可以用边缘按钮从上、下、左、右增减行列。
3. 点击格子启用或禁用对应杜邦母座焊孔。
4. 选中格子后填写引脚编号和名称。
5. 设置全局引脚名丝印位置、字号和偏移；也可以在当前格子或批量引脚丝印中覆盖指定引脚。
6. 使用“线 / 矩形 / 文本”模式绘制顶层丝印。
7. 选择器件行数、每行引脚数、镜像或翻转，右侧会实时预览专业版符号。
8. 右侧 PCB 预览会同步显示焊盘、钻孔、编号、外框和丝印。
9. 拖动左栏/中间栏、中间栏/右栏之间的分隔条，可以调整三栏宽度。
10. 优先点击“专业版完整器件库”，下载同时包含器件、符号和封装的 `.elibz2` 文件。

缩小高度或宽度时，如果已有启用格子位于更靠下或更靠右的位置，应用会自动保留能显示这些启用格子的最小行列数。

使用边缘按钮从上方或左侧增减行列时，已启用焊孔和手绘丝印会一起平移，保持它们在封装中的相对位置。删除边缘行列时，如果该边缘包含已启用焊孔，应用会拒绝删除。

## 器件样式

当前支持按行组织专业版符号引脚：

- `单行`：所有引脚作为一组从左侧引出
- `双行`：按启用顺序左右交替分配，适配 1-2-3-4 这类交叉编号
- `四行`：按启用顺序轮转分配到上、下、左、右四侧，适配 1-2-3-4 / 5-6-7-8 这类交叉编号
- `每行引脚数`：留空时自动平均分配，填写数字时按该数量分组，超出的引脚会继续追加行组
- `四行引脚数`：四行模式下可以分别设置上、下、左、右的引脚数量，留空的侧边自动分配剩余引脚
- `器件左右镜像 / 上下翻转`：只改变原理图符号的引脚分布方向，不改变 PCB 焊孔阵列

## PCB 预览

右侧 PCB 预览会显示焊盘、钻孔、引脚编号、自动外框丝印和手绘丝印。

`外框丝印距离 mm` 控制自动外框到最外侧焊盘中心的距离，并同步影响 PCB 预览和导出的专业版封装。

如果勾选 `分别设置外框四边距离`，可以分别填写上、下、左、右四个方向的距离；未勾选时四边使用统一距离。

`名称丝印字号 mm` 控制封装中器件名称文字的丝印字体大小，并同步影响 PCB 预览和导出的专业版封装。

`引脚名丝印` 可以设置为不显示、正上、正下、正左或正右，并支持字号、左右偏移和上下偏移。当前格子面板可以单独覆盖某个引脚的丝印样式；“批量引脚丝印”支持用 `1,2,5-8` 这类编号选择器一次设置多个已启用引脚。

## 自动缓存

应用会在浏览器中自动缓存当前项目数据：

- 每次编辑后会延迟写入 cookie 缓存，下次打开页面会自动恢复上次数据
- cookie 使用分片保存，支持比单个 cookie 更大的项目数据
- 如果浏览器在本地 `file://` 页面下限制 cookie，会自动使用 `localStorage` 作为兜底缓存
- “保存项目 JSON / 导入项目 JSON”仍然可用于手动备份和迁移

缓存内容包含网格尺寸、启用格子、引脚编号和名称、引脚名丝印设置、手绘丝印、符号样式、外框距离、镜像、翻转设置和三栏宽度。

## elibz2 文件结构

`.elibz2` 是 ZIP 容器，内部包含：

- `*.elibu`：专业版库记录流，包含 `SYMBOL` 和/或 `FOOTPRINT` 文档
- `device2.json`：完整器件库的器件、符号与封装 UUID 索引和关联关系
- `symbol2.json` / `footprint2.json`：单独符号库或封装库的 UUID 索引

在嘉立创EDA专业版中通过“文件 -> 导入 -> 嘉立创EDA（专业版）”导入 `.elibz2` 文件。完整器件库会把符号和封装绑定到同一个器件；两个单独导出按钮用于只需要符号或封装的场景。
